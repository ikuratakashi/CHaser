# CHaser client library - Python3

試合中にキー入力を求めて、それに応じて動作するようにしたプログラムです。
仮想の対戦相手として利用すると良いと思います。

run.sh がクライアントの起動シェルです。
ホストの設定に合わせて修正してください。

## 自分の環境での動かし方

COOLで起動する場合
```sh
python3.12 CHaserGame.py -c c
```

HOTで起動する場合
```sh
python3.12 CHaserGame.py -c h
```

## シェルの実行方法

COOLで起動する場合
```sh
 ./run.sh c
```

HOTで起動する場合
```sh
 ./run.sh h
```
